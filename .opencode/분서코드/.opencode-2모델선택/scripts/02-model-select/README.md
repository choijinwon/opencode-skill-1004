# 02-model-select 분석 메모

이 폴더는 `2. 모델 선택` 단계의 Python 스크립트를 따로 분석하기 위한 위치입니다.

원본 기준:
- `.opencode/scripts/02-model-select/select_model.py`

핵심 역할:
- 사용자가 입력한 모델 번호 또는 상대경로를 정규화합니다.
- 1차 분석에서 보여준 모델 목록 기준으로 번호를 실제 모델 경로로 변환합니다.
- 선택 모델을 이후 단계에서 계속 유지할 수 있도록 5차 준비 스크립트에 `select_only=True`로 위임합니다.
- 기본 출력에서는 선택 모델, `MODEL_KIND`, 실행 파일, 작업 폴더, 다음 가능 단계를 짧게 보여줍니다.

주의:
- 이 단계는 템플릿 변환을 직접 수행하지 않습니다.
- 이 단계는 원격 MLflow 등록을 실행하지 않습니다.
- 이 단계는 "선택 모델 고정"까지만 담당합니다.
